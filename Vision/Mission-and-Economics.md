# Mission and Economics

## The Problem
Lead gen operators building hundreds of local service websites need directory citations for:
1. Local SEO authority signals
2. Google Business Profile (GBP) verification corroboration
3. Direct directory visibility/traffic

## Why Existing Tools Fail
- **Yext**: $199-$999/yr per location → $100K-$900K/yr at 1,000 sites
- **BrightLocal**: $29-$79/mo per location → unviable at scale
- **Manual / VA**: Error-prone, no NAP consistency guarantee, can't scale to hundreds/month

## The Economic Case for Building In-House

| Scale | Commercial Tools | NAP Engine |
|-------|-----------------|------------|
| 100 sites | $14K-$83K/yr | ~$200/mo infra |
| 500 sites | $70K-$415K/yr | ~$200/mo infra |
| 1,000 sites | $140K-$900K/yr | ~$200/mo infra |

**Break-even: ~50-100 active sites.** Every site beyond that is pure savings.

## Value Chain
Citations → GBP Verification → Map Pack Presence → High-Intent Local Leads → Revenue

NAP is the first domino. Without corroborating web presence, GBP verification faces friction. Without GBP, no map pack. Without map pack, no high-intent traffic.

## Aggregator-First Architecture
Priority: Foursquare, Data Axle, Neustar/Localeze — each cascades to 100+ secondary directories.
3-4 aggregator submissions = 80% of citation value with minimal engineering.
