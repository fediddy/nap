# Phased Roadmap

## Tier 1 — Direct Free Submissions (MVP)
**Status**: Planned, partially built (Stories 1.1-1.5 complete)
- Direct API integrations: Bing Places, Facebook Business, Yelp
- Browser automation: Google Business Profile, Apple Maps (where free claim requires form)
- **Gate**: 10 sites submitted successfully across all directories

## Tier 2 — Browser Automation Expansion
**Status**: Not started
**Trigger**: 60-day Tier 1 impact validation (before committing)
- Playwright adapters for high-DA directories without APIs
- Camoufox anti-detection browser
- Proxy rotation infrastructure
- Each adapter = ongoing ~5-10 hrs/month maintenance commitment

## Tier 2b — Aggregator Partnerships
**Status**: Not started
**Trigger**: ROI data from Tier 1
- Paid feeds: Data Axle, Foursquare, Neustar/Localeze
- Each cascades to 100+ secondary directories automatically
- Cost: $30-80/listing — evaluate once Tier 1 impact measured

## Tier 3 — Niche Intelligence
**Status**: Not started
- AI-powered niche-specific directory selection
- Industry vertical directory database
- Roofing: GAF, Owens Corning, CertainTeed, NRCA, HomeAdvisor, Thumbtack
- Medical: Healthgrades, Zocdoc, WebMD
- Legal: Avvo, FindLaw, Justia
- Home services: Angi, Houzz, Thumbtack, Porch

## Phase 4 — Platform Integration
- Full pipeline: site creation → NAP → citations → GBP prep
- GBP management module (separate)
- API endpoint for AeliseIntelligence integration
