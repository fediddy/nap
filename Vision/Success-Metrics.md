# Success Metrics

## Operational Targets
| Metric | Target |
|--------|--------|
| Hands-on time per site | < 2 minutes (import to submission queued) |
| Batch capacity | 50+ sites per session |
| Submission success rate | > 90% Tier 1 directories |
| Daily monitoring time | < 5 minutes |
| Infrastructure cost | < $200/month |

## Milestone Gates
- **Launch gate**: 10 sites submitted to all integrated directories
- **60-day gate**: 50+ sites, dashboard accurate, daily check < 5 min
- **6-month**: 200+ sites, before/after impact data, Tier 2 go/no-go
- **12-month**: 500+ sites, cost/site < $0.50, citations automated in site launch pipeline

## Anti-Vanity Rule
"Submissions completed" is an OUTPUT metric, not a success metric.
Judge success by OUTCOMES: local search impressions, GBP verification rate, time-to-first-lead.

## Leading Indicators
- Citation coverage rate: % of target directories with verified listing per site
- NAP consistency score: % of listings with matching data across all directories
- Submission queue health: success/fail ratio per week
- Directory uptime: % of integrations functioning without intervention

## Lagging Indicators
- Local search impression delta (GSC: sites with citations vs without)
- GBP verification success rate on first attempt
- Time-to-first-lead (pre-NAP vs post-NAP cohorts)
