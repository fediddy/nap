# Target Users

## Primary: "The Lead Gen Operator" (Fediddy)

Solo operator running hundreds of local service websites across all niches. Managing the full pipeline: site creation → SEO → citation building → lead monetization — single-handedly.

**Technical skill**: Intermediate — comfortable with CSV, web dashboards, API concepts. Not maintaining complex infrastructure daily.

**Core pain**: Skips citations entirely because no viable way to do it at scale alone.

## Usage Modes

### Mode A: "The Batch Launcher"
New batch of 20-50 sites this week. Needs to import profiles, review submission plans, fire citations. Concentrated bursts — heavy import/submit → quiet periods. Speed and bulk efficiency are everything.

### Mode B: "The Monitor"
Between launches, quick daily check. What succeeded? What failed? What needs manual intervention? < 5 minutes. Scan dashboard, handle action queue, move on.

## The "Aha" Moment
Import 50 businesses → hit "submit" → walk away → check 48 hrs later → 250+ directory submissions completed automatically with 90%+ success rate. What was impossible is now a background process.

## Future: Secondary Users
Currently internal only. If operation scales to include clients or team, multi-tenant access to be revisited as Phase 4 feature.
