# KPI Tracking

## Operational KPIs (dashboard visible)
| KPI | Target | Where |
|-----|--------|-------|
| Submission success rate | > 90% | Per directory, per week |
| Queue depth | < 100 pending | Dashboard summary |
| Directory uptime | 100% | Directory health panel |
| Action queue age | < 24 hrs | Action queue view |

## Business Impact KPIs (manual measurement)
| KPI | Target | Method |
|-----|--------|--------|
| Local search impressions | +X% | Google Search Console comparison |
| GBP verification success rate | Track % | Manual log |
| Time-to-first-lead | Track delta | CRM comparison |
| Cost per citation profile | < $0.50 | Monthly infra cost / active sites |

## Anti-Vanity Rule (from Product Brief)
"Submissions completed" is an OUTPUT metric, not a SUCCESS metric.
- OUTPUT: submissions completed, directories covered
- SUCCESS: ranking improvements, GBP verification rate, lead volume

Track both but judge success only by outcomes.

## Measurement Cadence
- **Daily**: Queue health, action queue items, directory health
- **Weekly**: Submission success rate per directory
- **Monthly**: Business impact metrics, cost per site
- **60-day gate**: Tier 1 impact assessment → Tier 2 go/no-go
